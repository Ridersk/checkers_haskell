# Jogo de Damas

## Sobre

**Linguagens**: Haskell
**Tecnologias**: Gtk2

Jogo de Tabuleiro Simples

## Uso  

* Instalar Dependências do Sistema

      sudo apt install libgtk2.0-dev

* Build do Projeto

      cabal update
      cabal configure
      cabal build

* Rodar jogo

      cabal run
